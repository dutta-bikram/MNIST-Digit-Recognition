# MNIST Digit Recognition

A PyTorch-based neural network for handwritten digit classification.

## Dataset
MNIST (60,000 train, 10,000 test)

## Model
- Input: 28x28 image
- Hidden Layer 1: 128 neurons
- Hidden Layer 2: 64 neurons
- Output: 10 classes

## Accuracy
97.62%

## Technologies
- Python
- PyTorch
- Google Colab
